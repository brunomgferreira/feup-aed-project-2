void CityStatisticsMenuState::display ( ) const
overridevirtual

Displays the menu for city-specific statistics.

This method outputs a formatted menu to the console, presenting various statistical options related to the specified city. Users can choose from options such as the number of airports, cities, airlines, flights, destinations, and reachable destinations with a maximum number of stops.

void CityStatisticsMenuState::handleInput ( App *  app)
overridevirtual

Handles user input for city-specific statistics menu.

This method prompts the user for input and processes the chosen option for city-specific statistics. Users can choose from options such as the number of airports, countries, airlines, flights, destinations, and reachable destinations with a maximum number of stops. Additionally, users can navigate back to the statistics menu or return to the main menu.
