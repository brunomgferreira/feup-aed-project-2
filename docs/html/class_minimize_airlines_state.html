MinimizeAirlinesState::MinimizeAirlinesState ( function< void(App *, bool)>  nextStateCallback)

Constructs a MinimizeAirlinesState object with a callback for the next state.

This constructor initializes a MinimizeAirlinesState object with a callback function that will be executed in the next state. The callback function takes a pointer to the application instance and a boolean parameter indicating whether to minimize airlines or not. This state is typically used to prompt the user for minimizing or not minimizing airlines in the context of flight-related queries.

Displays a prompt for user input regarding airline minimization.

This method prints a prompt to the console, asking the user whether they want to show only flights with the minimum number of different airlines. The user is expected to respond with 's' for yes or 'n' for no.
