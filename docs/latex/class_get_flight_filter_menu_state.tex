Handles user input for selecting flight filter options. 

This method reads a user\textquotesingle{}s input from the console and processes it based on the chosen option. Users can input a single character corresponding to the desired option (1-\/3, \textquotesingle{}b\textquotesingle{} for Destination Menu, or \textquotesingle{}q\textquotesingle{} for Main Menu). The method then transitions to the appropriate state based on the user\textquotesingle{}s choice, capturing relevant information such as a set of airlines, a set of unwanted airlines, or no filter at all. The captured information is used to set up the next state for retrieving flights based on the selected filters. Error messages are displayed for invalid inputs, and the state may transition accordingly.
