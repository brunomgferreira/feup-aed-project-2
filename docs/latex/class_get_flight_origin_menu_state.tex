Displays the menu for selecting the type of origin for flight retrieval. 

This method prints the menu to the console, allowing users to choose the type of origin when retrieving flight information. The displayed menu includes options such as selecting an airport, city, country, coordinates, or coordinates with a radius. Users can input a corresponding numeric option, and the method visually enhances the menu with color formatting. The menu also provides an option to return to the main menu.

Handles user input for selecting the type of origin for flight retrieval. 

This method reads a user\textquotesingle{}s input from the console and processes it based on the chosen option. Users can input a single character corresponding to the desired option (1-\/5 or \textquotesingle{}q\textquotesingle{} for Main Menu). The method then transitions to the appropriate state based on the user\textquotesingle{}s choice, capturing relevant information such as airport code, city name, country name, coordinates, or coordinates with a radius. The captured information is used to set up the next state for flight destination selection. Error messages are displayed for invalid inputs, and the state may transition accordingly.
