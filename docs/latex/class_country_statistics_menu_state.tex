This method prints the menu for country-\/specific statistics to the console. Handles user input for country-\/specific statistics menu. 

This method reads a user\textquotesingle{}s choice from the console and performs the corresponding action based on the selected option. It allows the user to view the number of airports, cities, airlines, flights, and destinations for the specified country. Additionally, it provides an option to find reachable destinations with a maximum number of stops, return to the statistics menu, or go back to the main menu.
