Handles user input for obtaining an airport code. 

This method reads a line of input from the console, representing an airport code. The method validates the existence of the entered airport code in the application\textquotesingle{}s data. If the airport code is valid, the callback function is invoked with the captured airport code, and the application transitions to the next state accordingly. If the airport code is invalid, the user is prompted with an error message, and the state transitions to a \char`\"{}\+Try Again\char`\"{} state, allowing the user to make another attempt.
